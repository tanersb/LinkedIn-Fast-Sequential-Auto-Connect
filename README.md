# LinkedIn Fast Sequential Auto Connect

Ultra fast Tampermonkey userscript for sending LinkedIn connection requests sequentially.

## Features
- Sequential queue system
- Skips "Add note" popup
- Start / Stop floating panel
- Auto update from GitHub

## Install

Install Tampermonkey, then open:

https://raw.githubusercontent.com/tanersb/LinkedIn-Fast-Sequential-Auto-Connect/main/linkedin-fast-sequential.user.js

Tampermonkey will offer install.

## Usage

Open LinkedIn people list and press **START**.
